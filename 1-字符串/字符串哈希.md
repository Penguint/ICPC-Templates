## 字符串哈希

`hash(i) = hash(i - 1) * 33 + str[i];`

Another: `hash(i) = hash(i - 1) * 33 ^ str[i]`

### char[]
```c++
unsigned int hash(unsigned char *str) {
    unsigned int hash = 5381;
    int c;
    while (c = *str++)
        hash = ((hash << 5) + hash) + c; /* hash * 33 + c */
    return hash;
}
```
### string
```c++
constexpr unsigned int hash(string &s) {
    unsigned int hash = 5381;
    for (char c : s)
        hash = ((hash << 5) + hash) + c; /* hash * 33 + c */
    return hash;
}
```
