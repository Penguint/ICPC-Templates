## 蔡勒公式

输入年、月、日，计算格里高利历的星期。

有两种表示星期的方式：

 - Zeller

    |     |           |
    | --- | --------- |
    | 0   | Saturday  |
    | 1   | Sunday    |
    | 2   | Monday    |
    | 3   | Tuesday   |
    | 4   | Wendesday |
    | 5   | Thursday  |
    | 6   | Friday    |

 - ISO week date

    |     |           |
    | --- | --------- |
    | 1   | Monday    |
    | 2   | Tuesday   |
    | 3   | Wendesday |
    | 4   | Thursday  |
    | 5   | Friday    |
    | 6   | Saturday  |
    | 7   | Sunday    |

```cpp
int Zeller(int y, int m, int d) {
    if (m == 1 || m == 2)
        m += 12, y = y - 1;
    int h = (d + 13 * (m + 1) / 5 + y + y / 4 - y / 100 + y / 400) % 7;
    int w = (h + 5) % 7 + 1; // ISO week date (1 = Monday, 7 = Sunday)
    return w;
}
```
