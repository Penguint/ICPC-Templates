## 阶乘 Factorial


### 朴素实时
```cpp
int fac(int n, int mod = mod) {
    ll res = 1;
    for (ll i = 1; i <= n; i++)
        res = res * i % mod;
    return res;
}
```

### 打表
```cpp
ll _fac[1000000];
void make_fac(int n, int mod = mod) {
    _fac[0] = 1;
    for (ll i = 1; i <= n; i++)
        _fac[i] = _fac[i - 1] * i % mod;
}
inline int fac(int n) { return _fac[n]; }
```
