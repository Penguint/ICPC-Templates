## 阶乘的逆元 invfac

### 朴素实时
```cpp
// inv()
int invfac(int n, int mod = mod) {
    return inv(fac(n), mod);
}
```

### 打表
```cpp
// int inv()
ll _invfac[1000000];
void make_invfac(int n, int mod = mod) {
    _invfac[0] = 1;
    for (int i = 1; i <= n; i++)
        _invfac[i] = _invfac[i - 1] * inv(i) % mod;
}
inline int invfac(int n) { return _invfac[n]; }
```
