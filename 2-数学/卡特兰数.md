## 卡特兰数 Catalan number

通式：
$$
f(n) = \frac{C_{2n}^{n}}{n + 1}
$$

$$
f(n) = C_{2n}^{n} - C_{2n}^{n-1}
$$

递推式：
$$
f(n) = 
\begin{cases}
	\sum_{i=0}^{n-1}f(i)f(n-i-1) & n \ge 1 \\
	1 & n = 0
\end{cases}
$$



```c++
// int C(int n, int m);
ll Catalan(int n) { return (0ll + C(2 * n, n) - C(2 * n, n - 1) + mod) % mod; }
```
