## mod运算逆元 Inverse element

### mod p (p为大于2的质数) 朴素实时
```cpp
// qpow()
int inv(int n, int mod = mod) { return qpow(n, mod - 2); }
```

### mod p (p为大于2的质数) 打表
```cpp
ll _inv[1000000];
void make_inv(int n, int mod = mod) {
    _inv[1] = 1ll;
    for (int i = 2; i <= n; i++) {
        _inv[i] = _inv[mod % i] * (mod - mod / i) % mod;
    }
}
inline int inv(int n) { return _inv[n]; }
```
